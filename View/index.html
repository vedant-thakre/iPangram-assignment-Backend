Welcome to My API

This is a simple API serving JSON data. You can use the provided endpoints to interact with the data.

Endpoints:

  • GET /: Welcome page (this page)
  • GET /alldata: Get all data
  • GET /data/:id: Get data by ID
  • POST /create: Create new data
  • PUT /update/:id: Update data by ID
  • DELETE /delete/:id: Delete data by ID
